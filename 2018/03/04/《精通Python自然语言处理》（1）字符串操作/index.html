《精通Python自然语言处理》（1）字符串操作

2018-03-04
  1. sent_tokenize函数用于将文本切分为语句，该函数使用了NLTK包的PunktSentenceTokenizer类的实例。
  2. word_tokenize函数用于将句子切分为单词，该函数使用了NLTK包的TreebankTokenizer类的实例。
  3. 三个分词器：
    • TreebankWordTokenizer：通过分离缩略词实现切分
    • PunktWordTokenizer：通过分离标点实现切分
    • WordPunctTokenizer：通过将标点转化为一个全新的标识符实现切分
